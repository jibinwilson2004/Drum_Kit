## ЁЯЪА How to Run Locally

Since these are pure static projects, you don't need any complex build configurations or Node.js to run them!

1. Clone the repository:
   ```bash
   git clone https://github.com/jibinwilson2004/small_projects_for_beginners_using_Html-Css-Js.git
   ```
2. Open any project folder (e.g., `Drum_Kit`).
3. Simply double-click the `index.html` file to open it in your default browser, or use VS Code's **Live Server** extension for real-time updates!
